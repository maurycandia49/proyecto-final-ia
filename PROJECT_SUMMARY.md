## 🚀 Getting Started

### Prerequisites
- Node.js 16.0 or higher
- npm 7.0 or higher

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```
Opens at http://localhost:3000 with hot reload

### Production Build
```bash
npm run build
```
Creates optimized build in `dist/` folder

### Preview Production Build
```bash
npm run preview
```
